Fundamentos em CSS

1. Leia atentamente o guia de estudos e realize testes simulando as situações abaixo, e marque apenas as alternativas verdadeiras e justifique-as.

  • Uma mesma folha de estilos pode ser usada por várias páginas ✔️

    • Uma mesma folha de estilos (arquivo CSS) pode ser usada por várias páginas HTML. Isso promove a consistência visual e facilita a manutenção, pois as alterações no CSS se refletirão em todas as páginas que usam a mesma folha de estilos.

  • Uma mesma página pode usar várias folhas de estilo. ✔️

    • Uma mesma página HTML pode usar várias folhas de estilo CSS. Isso pode ser feito usando múltiplas tags link ou style no cabeçalho da página. No entanto, é importante observar a ordem de prioridade das regras CSS quando várias folhas de estilo são aplicadas a uma página.

  • Se um browser não suportar uma folha de estilos requerida pela página, poderá haver uma degradação na qualidade da apresentação mas nunca haverá perda de informação. ✔️

    • Se um navegador não suportar uma folha de estilos específica requerida por uma página, pode haver uma degradação na qualidade da apresentação, mas a perda de informação geralmente não ocorre. O conteúdo ainda é legível e acessível, mas pode não ser exibido exatamente como pretendido.

  • É possível construir um site inteiro usando apenas CSS. ✔️

    • É possível construir um site inteiro usando apenas CSS, mas isso seria incomum. O CSS é usado principalmente para controlar o estilo e o layout das páginas da web, enquanto o HTML é usado para criar a estrutura e o conteúdo. No entanto, o CSS é fundamental para o design web moderno.

  • A linguagem CSS usada para construir folhas de estilo é uma recomendação do W3C – consórcio de empresas que estabelece os padrões para a Web ✔️

    • A linguagem CSS (Cascading Style Sheets) é uma recomendação do W3C (World Wide Web Consortium), que é um consórcio de empresas que estabelecem padrões para a web. O W3C é uma autoridade em relação a padrões web, e o CSS é uma das especificações mantidas por eles para estilizar páginas web de forma consistente em diferentes navegadores.

2. No exemplo, Material de Aula – Arquivo Template, qual a forma utilizada para a inserção dos estilos na página index.html? Justifique sua resposta.

A forma utilizada para o template foi o modelo: "Folhas de Estilo Externas (External Stylesheets)". Este é o método mais comum e recomendado. É criado um arquivo de folha de estilo separado com extensão .css e, em seguida, é vinculado esse arquivo ao documento HTML usando a tag link no cabeçalho do documento.
